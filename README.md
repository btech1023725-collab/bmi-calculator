# BMI Calculator

A simple and modern BMI (Body Mass Index) calculator built using HTML, CSS, and JavaScript.

## Features

* Metric & Imperial units
* BMI category detection
* Ideal weight calculation
* Health tips based on BMI
* Clean and responsive UI

## How to use

1. Enter your height and weight
2. Click "Calculate BMI"
3. View your result instantly
